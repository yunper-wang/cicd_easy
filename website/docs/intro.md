# cicd_easy — CI/CD 交互式学习

cicd_easy 是一个 **从零到生产级** 的 CI/CD GitOps 交互式学习平台。

通过 3 个渐进式 Stage，你将掌握 GitLab CI + Argo CD + Argo Rollouts 的完整 GitOps 工作流。

## 为什么选择 cicd_easy？

- **交互式学习** — 通过可视化配置、流程模拟、代码对比理解每个概念
- **渐进式设计** — 从静态部署到多环境管理再到 Canary 发布，层层递进
- **真实技术栈** — 基于 GitLab CI、Argo CD、Argo Rollouts、Kubernetes 等行业标准工具
- **透明对照** — 每个配置操作都对应真实的 CI/CD 流程，学习过程即实战过程

## 3 个学习阶段

| Stage | 主题 | 核心技能 |
|-------|------|----------|
| **Stage 1** | 静态部署 | Docker 镜像构建 + GitLab CI Pipeline + Argo CD 自动同步 |
| **Stage 2** | 多环境管理 | Kustomize base/overlay + dev/staging/prod 差异化配置 |
| **Stage 3** | Canary 发布 | Argo Rollouts 金丝雀策略 + AnalysisTemplate + 自动回滚 |
