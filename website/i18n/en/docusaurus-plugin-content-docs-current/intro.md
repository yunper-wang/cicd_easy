# Welcome to cicd_easy

> Making CI/CD learning simple and transparent.

**cicd_easy** is an interactive learning platform that takes you from zero to production-grade CI/CD using GitOps workflows.

## 3 Progressive Stages

| Stage | Topic | Level |
|-------|-------|-------|
| **Stage 1** | Static Deployment | L1 Beginner |
| **Stage 2** | Multi-Environment Management | L2 Intermediate |
| **Stage 3** | Canary Release | L3 Production |

## What You'll Learn

- **GitOps principles** — Declarative, version-controlled infrastructure
- **GitLab CI** — Build, test, and deploy pipelines
- **Argo CD** — GitOps continuous delivery
- **Argo Rollouts** — Canary and blue-green deployment strategies
- **Kustomize** — Environment-specific configuration management

## Quick Start

```bash
git clone https://github.com/yunper-wang/cicd_easy.git
cd cicd_easy
./scripts/setup.sh
```

Or explore the [interactive learning website](/docs/getting-started) directly.
